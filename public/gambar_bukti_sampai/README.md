# Folder Gambar Bukti Sampai

Folder ini digunakan untuk menyimpan gambar bukti pengiriman furniture yang telah sampai ke customer.

## Struktur Penamaan File:
- Format: `bukti_{id_penjualan}_{timestamp}.{ekstensi}`
- Contoh: `bukti_1_1701234567.jpg`

## Spesifikasi:
- Format: JPG, PNG
- Ukuran maksimal: 5MB per file
- Resolusi minimum: 480x640px

## Penggunaan:
Diupload oleh customer sebagai konfirmasi bahwa barang telah diterima dengan baik.
